react-native-安装流程

工具安装

  • Python @2.7.13
  • node -> react-native-cli
  • 安卓 JDK jdk-8u131-windows-x64
  • android-studio

–安装Python @python-2.7.13.amd64:

下载 python-2.7.13版本 时间 2017 06-11

–安装node、yarn、react-native-cli:

下载 node 命令行安装 $ npm install -g yarn react-native-cli

–安装安卓的JDK → jdk-8u131-windows-x64.exe:

下载 安卓JDK 1.版本以上

–安装android-studio-bundle-162.4069837-windows:

下载 android-studio 下载2.0以上版本

android-studio 安装

安装流程截图:

记下 SDK 的安装路径

取消安装SDK

出现以下界面 点击右下角的 Configure 设置SDK

勾选3个必选的SDK



配置系统环境变量 在我的电脑属性 新建环境变量
变量名 ANDROID_HOME
变量值 C:\Users\JackieZon\AppData\Local\Android\Sdk 这是本台机的SDK路径

在 Pathl里添加两个环境变量

完成

安卓模拟器下载地址 genymotion

初始化Android 项目

  1. react-netive init helloWorld
  2. 运行安装项目
  3. react-netive run-android
